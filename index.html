Responsive & Media Queries
Adapter l'affichage selon la taille d'écran.
C'est quoi le responsive ?+

Adapter le design du site à la taille de l'écran. Mobile First : d'abord pour mobile, puis on enrichit pour les grands écrans.

Breakpoints principaux+
RésolutionCible
< 480/576pxMobile
≥ 768pxTablette
≥ 960pxDesktop
Syntaxe @media+
@media screen and (max-width: 576px) { /* mobile */ } @media screen and (min-width: 768px) { /* tablette */ } @media screen and (min-width: 960px) { /* desktop */ }

max-width = jusqu'à cette taille. min-width = à partir de cette taille.

Viewport — balise meta indispensable+
<meta name="viewport" content="width=device-width">

Sans elle, le navigateur mobile dézoome et ignore les media queries. CSS pixel ratio : 1px CSS ≠ 1px physique.

CSS Grid Layout
Mise en page bidimensionnelle. Conteneur + enfants.
Activer la grille + fr + repeat()+
.conteneur { display: grid; grid-template-columns: repeat(3, 1fr); gap: 16px; } /* Responsive sans media query */ grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
span, grid-column, nth-child+
/* 2 colonnes */ .el { grid-column: span 2; } /* de col 1 à 3 */ .el { grid-column: 1 / 3; } /* 3e enfant */ li:nth-child(3) { grid-column: span 2; }
grid-area — zones nommées+
.conteneur { grid-template-areas: "header header" "main aside" "footer footer"; } header { grid-area: header; } main { grid-area: main; }
TP Calculatrice — structure Grid+
#calc-grid { display: grid; grid-template-columns: repeat(4, 100px); grid-template-rows: minmax(120px, auto) repeat(5, 100px); gap: 4px; } .display { grid-column: span 4; } .span-two { grid-column: span 2; }
