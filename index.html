Responsive & Media Queries
Adapter l'affichage selon la taille d'écran.
C'est quoi le responsive ? +

Adapter le design du site à la taille de l'écran de l'utilisateur. On distingue Mobile, Tablette, PC, TV.

Mobile First : on développe d'abord pour le mobile (le plus contraignant), puis on enrichit pour les grands écrans.

Breakpoints principaux +
RésolutionCible
< 480 / 576pxMobile
≥ 768pxTablette
≥ 960pxDesktop
Syntaxe @media +
@media screen and (max-width: 576px) { /* styles mobile */ } @media screen and (min-width: 576px) { /* styles tablette */ } @media screen and (min-width: 960px) { /* styles desktop */ }

max-width = "jusqu'à cette taille". min-width = "à partir de cette taille".

Le viewport — balise meta indispensable +

Les mobiles dézooment par défaut pour afficher le site. Sans la balise viewport, tes media queries ne fonctionneront pas sur mobile.

<meta name="viewport" content="width=device-width">

Le CSS pixel ratio : un pixel CSS ≠ un pixel physique. Un iPhone Retina a un ratio de 2 (640px physiques = 320px CSS).

Règles @media disponibles +
  • width / height — largeur/hauteur du viewport
  • orientation — portrait ou landscape
  • media screen / print — type d'affichage
  • prefers-color-scheme — mode clair/sombre
  • device-width / device-height — dimensions physiques
CSS Grid Layout
Mise en page bidimensionnelle. Conteneur + enfants.
Activer la grille +
.conteneur { display: grid; grid-template-columns: 1fr 1fr 1fr; /* 3 colonnes égales */ grid-template-rows: 100px 100px; /* 2 lignes de 100px */ gap: 16px; /* gouttière */ }
L'unité fr et repeat() +

fr = fraction de l'espace disponible.

/* 3 colonnes égales */ grid-template-columns: 1fr 1fr 1fr; /* Raccourci avec repeat() */ grid-template-columns: repeat(3, 1fr); /* Mix : colonne fixe + flexible */ grid-template-columns: 200px repeat(3, 1fr); /* Auto-remplissage responsive */ grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
minmax() — taille min et max +
/* Colonne entre 100px et 300px */ grid-template-columns: minmax(100px, 300px); /* Ligne : min 120px, s'adapte au contenu */ grid-template-rows: minmax(120px, auto);

Utilisé dans le TP calculatrice pour l'écran d'affichage : minmax(120px, auto).

Placer les éléments — span, grid-column, grid-row +
/* Occuper 2 colonnes d'affilée */ .mon-element { grid-column: span 2; } /* Occuper 4 colonnes : calculatrice display */ .display { grid-column: span 4; } /* Placement précis */ .element { grid-column: 1 / 3; /* de la ligne 1 à 3 */ grid-row: 2 / 4; } /* nth-child — sélectionner le 5e élément */ li:nth-child(5) { grid-column: 2 / 4; }
Gouttières (gap) +
gap: 16px; /* lignes et colonnes */ row-gap: 10px; /* entre les lignes seulement */ column-gap: 20px; /* entre les colonnes seulement */
Zones nommées — grid-area +
/* Sur le conteneur */ .conteneur { display: grid; grid-template-areas: "header header" "section aside" "footer footer"; } /* Sur chaque enfant */ header { grid-area: header; } section { grid-area: section; } aside { grid-area: aside; } footer { grid-area: footer; }

TP Calculatrice — structure Grid +

Grille 4 colonnes × 6 lignes. Colonnes et lignes de 100px. Affichage : minmax(120px, auto).

JavaScript — DOM & Events
Variables, fonctions, sélecteurs, événements, JSON.
Variables et constantes +
const nom = "Mario"; // ne change jamais let age = 39; // peut changer console.log(nom + " a " + age + " ans"); // → Mario a 39 ans
  • Commence par une lettre ou _, pas de chiffre ni caractère spécial
  • camelCase pour les variables : nomDeLaVariable
  • SCREAMING_SNAKE_CASE pour les constantes importantes
Sélecteurs DOM +
// Par ID → un seul élément const btn = document.getElementById('mon-btn'); // Par classe → plusieurs éléments const items = document.getElementsByClassName('item'); // Par sélecteur CSS → un seul const el = document.querySelector('.ma-classe'); const el2 = document.querySelector('[data-output]'); // Par sélecteur CSS → tous const tous = document.querySelectorAll('[data-number]');
data-* — attributs de données +
/* HTML */ <button data-number>1</button> <div data-output></div> <button data-correct="true">Bonne réponse</button> /* JS — sélectionner par data-* */ const nums = document.querySelectorAll('[data-number]'); const output = document.querySelector('[data-output]'); /* Lire/écrire une valeur data-* */ button.dataset.correct // lire button.dataset.correct = true; // écrire

Préféré aux classes CSS pour les rôles fonctionnels. Ne mélange pas sémantique CSS et sémantique JS.

TP Calculatrice — logique JS complète +
/* Sélecteurs */ const output = document.querySelector('[data-output]'); const clearButton = document.querySelector('[data-all-clear]'); const numberButtons = document.querySelectorAll('[data-number]'); /* Effacer */ function clear() { output.innerText = ''; } clearButton.addEventListener('click', () => { clear(); }); /* Afficher un chiffre */ function appendNumber(number) { output.innerText = output.innerText + number; } numberButtons.forEach(button => { button.addEventListener('click', () => { appendNumber(button.innerText); }); });

Rôles à attribuer : data-output, data-number, data-operation, data-equals, data-all-clear.

TP Quiz — logique JS complète +
let currentQuestionIndex = 0; function startGame() { currentQuestionIndex = 0; startButton.classList.add('hide'); questionContainer.classList.remove('hide'); setNextQuestion(); } function setNextQuestion() { resetState(); showQuestion(questions[currentQuestionIndex]); } function showQuestion(question) { questionElement.innerText = question.enonce; question.reponses.forEach(reponse => { let btn = document.createElement('button'); btn.innerText = reponse.texte; btn.classList.add('btn'); if (reponse.correcte) btn.dataset.correct = true; btn.addEventListener('click', selectAnswer); answerButtonsElement.appendChild(btn); }); } function selectAnswer(e) { const selectedButton = e.target; const isCorrect = selectedButton.dataset.correct; setStatusClass(document.body, isCorrect); Array.from(answerButtonsElement.children).forEach(button => { setStatusClass(button, button.dataset.correct); }); if (questions.length > currentQuestionIndex + 1) { nextButton.classList.remove('hide'); } else { startButton.innerText = 'Recommencer'; startButton.classList.remove('hide'); } } function setStatusClass(element, isCorrect) { element.classList.remove('correct', 'wrong'); element.classList.add(isCorrect ? 'correct' : 'wrong'); } function resetState() { answerButtonsElement.innerHTML = ''; nextButton.classList.add('hide'); }
